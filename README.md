# Extract-Text-From-PDF

This is the Open-Source contribution to public to use. It is very fast and easy way to extract text from pdf file. It was created to solve the general upcoming problem of text extraciton using Python script.

Accuracy Level maintain upto 80%. 

## Features
1. Very less code required.
2. Extract Text from any page number.
3. Extract Text between any page number.
4. All pdf
